# vps_code_and_db_backup
python script autobackup code and database of VPS 

1/ Create sftpinfo.txt in the same folder of the main script backup.py file with the information as bellow
>     mysql_root_username
>     mysql_root_password
>     mysql_host
>     sftp_hostname
>     sftp_username
>     sftp_password
>     sftp_port

## for example: ##
>     root
>     mypassword
>     localhost
>     111.111.111.111
>     222.222.222.222
>     mysftpusername
>     mysftppassword
>     mysftpport
